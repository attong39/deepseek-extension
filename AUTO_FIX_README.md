# Auto-Fix Missing Imports & Dependencies

🔧 **Sistema tự động phát hiện và sửa import thiếu + dependencies cho Python/TypeScript**

## Features

### ✨ Smart Symbol→Package Mapping
- **Python**: `ast` analysis + `importlib.metadata` để map module→distribution
- **TypeScript**: Regex detection + file search cho local components, fallback npm packages
- **Extensible**: `tools/auto_fix/symbol_to_pkg.json` cho custom mappings

### 🎯 Auto-Injection
- **Python**: Chèn `import <symbol>` vào đầu file, update `requirements.txt`
- **TypeScript**: Chèn `import Symbol from "path/pkg"`, update `package.json`
- **Smart**: Detect relative paths vs npm packages

### 📊 Unified Reporting
- **JSON**: `reports/auto_fix/report.json` cho CI/tooling
- **Markdown**: `reports/auto_fix/report.md` cho dev review
- **Exit codes**: 0=success, 2=unresolved symbols

## Usage

### CLI
```bash
# Fix tất cả
make fix-imports

# Chỉ Python
make fix-imports-py  # python tools/auto_fix/cli.py python

# Chỉ TypeScript  
make fix-imports-ts  # python tools/auto_fix/cli.py ts

# Dry run (không thay đổi)
make fix-imports-dry
```

## Configuration

### Symbol Mapping
`tools/auto_fix/symbol_to_pkg.json`:
```json
{
  "pandas": {"python": "pandas"},
  "React": {"ts": "react"},
  "lodash": {"ts": "lodash", "python": ""}
}
```

### Exit Codes
- **0**: All symbols resolved và fixed
- **2**: Có unresolved symbols (warning, không block CI)
- **1**: Tool error
