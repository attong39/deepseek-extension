## 🎮 Quick Start

1. **Open chat**: `Ctrl+Shift+I`
2. **Try một feature request**:
   ```
   @workspace Tạo một UserProfile entity hoàn chỉnh với:
   - Domain entity với validation
   - Repository interface
   - Application service với CRUD
   - FastAPI endpoints
   - Pydantic schemas
   - Unit tests
   Tuân thủ Clean Architecture patterns hiện tại.
   ```

3. **Use inline help**: `Ctrl+Alt+I` trên code cần fix
4. **Generate tests**: `Ctrl+Shift+A` cho test generation
5. **Get explanations**: `Ctrl+Alt+E` cho code explanation
