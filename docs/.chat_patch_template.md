# Patch Template Example

Sử dụng format sau để áp patch an toàn:

```zeta-file path="zeta_vn/example_module.py" lang="python" full="true"
# ZETA_FULLFILE:BEGIN
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""Example module demonstrating patch format."""

from __future__ import annotations

def example_function() -> str:
    """Example function with proper typing."""
    return "This is an example"

class ExampleClass:
    """Example class following project standards."""

    def __init__(self, name: str) -> None:
        """Initialize with name."""
        self.name = name

    def get_greeting(self) -> str:
        """Get formatted greeting."""
        return f"Hello, {self.name}!"

# ZETA_FULLFILE:END
```

## Usage Instructions

1. Copy your patch content to `.chat_patch.txt`
2. Run task: "Patch: Apply from .chat_patch.txt (interactive)"
3. Type "CONTINUE" to apply patches
4. Review diff and commit changes

## Safety Features

- Only allows writing to approved directories (zeta_vn/, desktop_ai_zeta/, etc.)
- Rejects bad filenames (`_copy`, `_final`, `backup`, `tmp`, etc.)
- Requires ZETA_FULLFILE markers for verification
- Prompts for confirmation before writing files
